# Install (MacOS)

Open Extensions `Ctrl+X` in Visual Code then type `C#` in the search field and install C# plugin `C# for Visual Studio Code (powered by OmniSharp)` by Microsoft.

Install `dotnet`:
```bash
brew install dotnet
```

Create the project:
```bash
dotnet new console -o csharp
cd csharp
```

Build the project:
```bash
dotnet build
```

Run the project:
```bash
dotnet run
```
